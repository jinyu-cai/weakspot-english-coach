## Safety boundaries

- Do not expose API keys, cookies, OAuth values, account identifiers, IP
  addresses, private learner content, or raw memory details.
- “Voice” means confirmed speech-to-text input and AI audio playback; do not
  claim acoustic pronunciation scoring.
- Picture Story uses first-party allowlisted illustrations.
- Listen & Retell uses an original generated passage or an authorized transcript.
